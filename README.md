
# Pulse-Guard

Site desenvolvido em React para o projeto Pulse-Guard


## Rodar localmente

Clone o projeto

```
  git clone https://link-do-projeto
```

Vá até o projeto

```bash
  cd projeto
```

Execute o site

```
  npm run dev
```

Execute a api local

```
  npm run start:api
```

Basta clicar no link localhost e aproveitar o site :)
